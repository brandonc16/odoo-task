## ARCHITECTURE 

The project  is provided mock-up into a resposive static webpage using HTML , JS , Bootrstrap 5.3, also the project uses native web components 
to organize the page into reusable UI section without using a framework.


## Running the project 
The project needs to be served through a local web server 
 we have two options: 

 Option 1 (VS code live server):
 1. install the Live server  extention in vs code
 2. Right click on index.html
 3. Click open whith live server 

 Option 2 (Using Node.js):
 
 Run the following command in the project folder:
 - npx serve
 
then open : http://localhost:3000
